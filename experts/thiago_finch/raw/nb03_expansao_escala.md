## O Modelo de Escala Digital

Thiago Finch ensina que escala é um processo matemático e sistemático:

```
FASE 1: Validação (R$0 → R$10k/mês)
   Produto validado
   Funil funcionando
   CAC < LTV

FASE 2: Otimização (R$10k → R$100k/mês)
   Aumentar conversão
   Reduzir custo por lead
   Teste de criativos

FASE 3: Escala (R$100k/mês → escala total)
   Aumentar budget
   Novos canais
   Afiliados
   Expansão de catálogo
```

---

## Tráfego Pago: A Base da Escala

### Meta Ads (Facebook/Instagram)
- Audiências: Lookalike, interesses, remarketing
- Tipos de anúncio: carrossel, vídeo, imagem estática, reels
- Estrutura de campanha: CBO (Campaign Budget Optimization)
- Escala: horizontal (mais conjuntos) + vertical (mais budget)

### Google Ads
- Search: palavras-chave de intenção
- Display: remarketing, branding
- YouTube Ads: vídeo in-stream

### Modelo de escalada de budget
```
Dia 1: R$100/dia (testar)
Semana 1: R$300/dia (validar)
Semana 2: R$1.000/dia (escalar)
Mês 2: R$3.000/dia (multiplicar)
```

## Funis Automatizados

### Funil Básico (Entrada)
```
Anúncio → Landing Page → Obrigado → Email Seq. → Oferta
```

### Funil Webinar (Conversão Alta)
```
Anúncio → Inscrição Webinar → Webinar Ao Vivo → Oferta → Replay → Encerra
```

### Funil Perpétuo (Escala)
```
Anúncio → Captura → Email Seq. → Página de Vendas → Oferta → Upsell
```

## Criativos que Escalam

Estrutura de criativos que funcionam:

### Vídeo (formato mais convertido)
```
0-3s: Hook forte (captura atenção)
3-10s: Problema (identidade)
10-30s: Agitação (pain points)
30-60s: Solução (promessa)
60-90s: Prova (resultados, depoimentos)
90s+: CTA direto (próximo passo)
```

### Imagem
```
Headline forte + Prova visual + CTA
```

## Afiliados como Canal de Escala

Ter afiliados multiplica o negócio sem aumentar custo fixo:

- **Seleção**: influenciadores e experts com audiência qualificada
- **Comissão**: 30%-50% em produtos de info
- **Materiais**: criativos prontos, copies, páginas validadas
- **Gestão**: Hotmart gerencia pagamentos automaticamente

### Resultado esperado
```
10 afiliados × R$10k cada = R$100k adicional sem custo de ads
```

## Expansão de Catálogo

Escala não é só mais tráfego — é mais produtos:

| Fase | Produto | Ticket |
|------|---------|--------|
| Entrada | Lead magnet / produto barato | Grátis - R$97 |
| Core | Curso principal | R$297 - R$997 |
| Premium | Mentoria em grupo | R$3.000 - R$10.000 |
| Top | Mentoria individual / mastermind | R$20.000+ |

### Value Ladder (Escada de Valor)
```
Free → R$97 → R$497 → R$2.000 → R$10.000 → R$50.000
```

## Métricas de Escala

| Métrica | Meta para Escalar |
|---------|-------------------|
| ROAS | > 3x (mínimo) |
| CPL (custo por lead) | Depende do ticket |
| Taxa de conversão página | > 2% |
| Taxa de abertura email | > 25% |
| CTR anúncio | > 1,5% |

---

## Quando Escalar

Regra de Thiago Finch:

> "Só escale quando a matemática fechar. Se você investe R$1 e sai R$3, escale. Se não fecha, não escale."

**Pré-requisitos antes de escalar:**
- Funil testado e convertendo
- Métricas estáveis por 7+ dias
- Suporte preparado para volume
- Produto entregando resultado

## Expansão Geográfica e Nicho

- **Mercado BR**: português, cultura local, plataformas nacionais
- **Mercado PT**: mesmo idioma, adaptação cultural mínima
- **Mercado US**: escala máxima, exige inglês e adaptação
