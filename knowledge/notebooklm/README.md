# NotebookLM Knowledge Base

Use esta pasta como sua base local para ingestão de conteúdo vindo do NotebookLM.

Fluxo recomendado:

1. Exporte/resuma materiais do NotebookLM em Markdown (`.md`).
2. Salve os arquivos aqui, em subpastas por tema, por exemplo:
   - `knowledge/notebooklm/marketing/`
   - `knowledge/notebooklm/produto/`
   - `knowledge/notebooklm/vendas/`
3. Sempre que criar conteúdo novo, prefira:
   - título objetivo
   - bullets acionáveis
   - seção de conclusões/recomendações

Padrão sugerido de nome:

- `YYYY-MM-DD_topico-principal.md`

Exemplo:

- `2026-02-25_estrategia-trafego-pago-b2b.md`
