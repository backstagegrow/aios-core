## Resumo do Tema Principal
O notebook foca no desenvolvimento técnico e automação dentro do ecossistema da Meta (Facebook, Instagram, etc.). O tema principal aborda a integração de APIs de Marketing, automação de processos publicitários e os requisitos rigorosos de segurança e conformidade da plataforma.

## 5 Insights Mais Importantes

1. **Migração Obrigatória para APIs Profissionais:** Em dezembro de 2024, a Meta desativou a *Instagram Basic Display API*. Agora, qualquer integração deve obrigatoriamente utilizar contas profissionais e migrar para a *Instagram Graph API* ou a nova *Instagram API com Login*.
2. **Tokens de Usuário do Sistema para Estabilidade:** Para automações estáveis que não dependem de intervenção humana, o uso de *Tokens de Usuário do Sistema (System User Tokens)* é essencial, pois eles podem ser configurados para nunca expirar, ao contrário dos tokens comuns (60 dias).
3. **Rigor no App Review e Verificação:** A aprovação de aplicativos exige gravações de tela (*screencasts*) detalhadas que provem o uso legítimo de permissões sensíveis (como `ads_management`), além da obrigatoriedade de verificação da empresa.
4. **Otimização com Webhooks e Batch Requests:** Para evitar limites de volume (*rate limits*), deve-se priorizar o uso de *Webhooks* para notificações em tempo real e *Batch Requests* para agrupar múltiplas operações em uma única requisição HTTP.
5. **Automação de Criativos com IA:** A integração do *Business SDK* da Meta com modelos como DALL-E 3 permite automatizar todo o ciclo de anúncios: desde a geração programática da imagem até o upload e criação da campanha completa sem intervenção manual.
