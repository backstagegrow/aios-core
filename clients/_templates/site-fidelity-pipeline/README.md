# Site Fidelity Pipeline (Template Generico)

Objetivo: reutilizar o fluxo `Stitch > AiStudio > Antigravity` para qualquer cliente sem acoplamento.

## Como usar
1. Copie esta pasta para o cliente/projeto alvo.
2. Preencha `stitch-to-aistudio-handoff.md` com dados reais.
3. Rode o prompt de `prompt-aistudio.md` no AiStudio.
4. Passe os artefatos gerados para `prompt-antigravity.md` no Antigravity.
5. Abra `story-template.md`, ajuste ID/escopo e acompanhe checklist.

## Estrutura
- `story-template.md`
- `stitch-to-aistudio-handoff.md`
- `prompt-aistudio.md`
- `prompt-antigravity.md`

## Observacoes
- Template neutro para multi-cliente.
- Sem dependencia de ViaBrCenografia.
- Pode ser versionado e evoluido por campanha.
