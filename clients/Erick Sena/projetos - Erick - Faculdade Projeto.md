## 📌 Pontos Mapeados (Lista de Atividades)

1.  **Mapeamento de Fontes de Dados:**
    *   [ ] **Currículo Lattes:** Link fixo por professor (Público).

2.  **Desenvolvimento do MVP (Core SaaS):**
    *   [ ] Configurar **Supabase** (Database SQL + Auth + Storage).
    *   [ ] Desenvolver **Frontend Dashboard** (Next.js/React).
    *   [ ] Criar sistema de **Login/Role Management** (Coordenação vs Professores).

3.  **Módulo de Inteligência & Automação:**
    *   [ ] Automação de extração de dados do **Lattes** (Índice H, Artigos, FWCI).

    *   [ ] Sistema de **Notificação de Prazos** (Jubilamento/Renovação de Bolsas).

4.  **Automações de Site e Divulgação:**
    *   [ ] Criar endpoint para **postagem automática de Bancas de Defesa** no site UFOP.
    *   [ ] Integração de divulgação automática no **Instagram** (via API Meta).

5.  **Indicadores CAPES & dashboards:**
    *   [ ] Dashboard de **Articulação e Avaliação**.
    *   [ ] Dashboard de **Produtividade Técnica** (Publicações, Journals Scopus).
    *   [ ] KPIs de **Tempo Médio de Defesa/Entrega**.

6.  **Governança & Formalização:**
    *   [ ] Registro formal do software na **UFOP**.
