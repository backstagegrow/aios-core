# Templates

Modelos reutilizáveis para standardizar research, discoveries e análises.

## Disponíveis

### Research
- **research-template.md** — Template para research estruturada
  - Use para: market analysis, competitive intel, product validation
  - Inclui: Resumo, Dados, Insights, Validação

### Discoveries
- **discovery-log.md** — Template para documentar descobertas
  - Use para: padrões identificados, novos insights, validações
  - Inclui: Contexto, Padrão, Implicações, Ações

### Analysis
- **offer-analysis-template.md** — Template para análise de ofertas
  - Use para: novo design de oferta, análise competitiva
  - Inclui: Spec, Mercado, Copy, Validação, Lançamento

## Como Usar

1. Copie template para pasta apropriada
2. Renomeie para seu tópico
3. Preencha frontmatter (created, author, etc)
4. Siga seções do template
5. Adicione backlinks para conectar
6. Commit quando concluído

## Customização

Templates podem ser customizados conforme necessário. Atualize nesta pasta e compartilhe melhoria com time.
