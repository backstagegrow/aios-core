# Discoveries

Insights e padrões descobertos durante execução de projetos.

## Estrutura

- **aba-insights.md** — Insights específicos do AlphaBusinessAcademy
- **client-patterns.md** — Padrões recorrentes com clientes
- **offer-learnings.md** — Aprendizados de ofertas lançadas

## Como Contribuir

1. Use template em `../templates/discovery-log.md`
2. Document com contexto claro
3. Tag com #discovery + categoria (#client, #offer, etc)
4. Adicione validação/replicação
5. Link para analysis/ quando relevante
6. Commit quando concluído

## Ciclo de Discovery

```
1. Observação durante projeto
2. Documentação com template
3. Identificação de padrão
4. Validação em próximo projeto
5. Incorporação em guidelines
```
