# 🏛️ Conselho dos Sábios — Sessões

Registro de todas as conversas com o Conselho Pessoal dos Sábios.

Cada sessão é um arquivo nesta pasta com data, tema e as respostas dos sábios.

---

## 📂 Sessões

> As sessões são salvas aqui automaticamente pelo comando `*save` durante uma conversa com `@conselho-dos-sabios`.

---

## 🧭 Como Usar

Durante uma sessão com o Conselho, ao final use:

```
*save [título opcional]
```

O agente criará um arquivo `YYYY-MM-DD-titulo.md` nesta pasta com toda a sessão.
