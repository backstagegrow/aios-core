# Research

Análises estruturadas de mercado, concorrência e produtos.

## Estrutura

- **market-analysis.md** — Tendências e tamanho de mercado
- **competitor-intel.md** — Análise de competidores
- **product-validation.md** — Validação de conceitos de produtos
- **trend-analysis.md** — Tendências emergentes

## Como Contribuir

1. Use template em `../templates/research-template.md`
2. Organize por categoria
3. Adicione tags (#market, #competitive-intel, etc)
4. Adicione backlinks para conexões
5. Commit quando validado
