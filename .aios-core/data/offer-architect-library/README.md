## 🎯 Como Usar com @offer-architect

### Ideação de Ofertas
```
1. Use EBOOK 30 IDEIAS para inspiração
2. Valide conceito com Checklist da Oferta
3. Use headlines templates para messaging
4. Aplique estrutura D6D para a página
```

### Construção de Copy
```
1. Revise 50 MODELOS DE COPY
2. Adapte headlines de 300 headlines
3. Use 30 COMANDOS CHATGPT para gerar variações
4. Valide com Checklist
```

### Documentação de Ofertas
```
1. Use template Modelo - Ficha Técnica
2. Preencha conforme especificação
3. Revise com Checklist da Oferta
4. Entregue aos stakeholders
```

### Briefing com Clientes
```
1. Selecione briefing apropriado (site, eventos, ads, etc)
2. Customize conforme projeto
3. Colete informações do cliente
4. Use como base para strategy
```

### Contratos
```
1. Selecione contrato apropriado (design, site, social, branding)
2. Customize com termos específicos
3. Revise com legal se necessário
4. Apresente ao cliente
```
