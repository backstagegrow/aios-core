# AIOS Digital Twin - NotebookLM Source

This directory contains consolidated knowledge sources intended for import into Google NotebookLM.
By using the AIOS Digital Twin, you can provide the AI with a deep understanding of:

1. **BKS Culture:** Strategic briefings and modeling.
2. **Copy Elite:** Successful patterns and winning copies.
3. **Growth Strategy:** Traffic and CRM integration logic.

## Usage

Run the following command to dispatch updates to your Digital Twin:

```bash
npm run notebooklm:dispatch
```

## Structure

- `/sources`: Raw files for import.
- `/context`: Consolidated context documents.
- `/manifest.json`: Tracking of synced files.
