## 6. Observações de UX

| Ponto | Recomendação |
|-------|-------------|
| **Mobile first** | 70%+ do tráfego virá mobile — hero com CTA visível sem scroll |
| **WhatsApp flutuante** | Botão fixo no canto inferior direito em todas as páginas |
| **Vídeo no hero** | Substituir imagem estática por loop de 6s de sorriso sendo transformado |
| **Antes/depois interativo** | Slider drag para comparação — aumenta tempo de permanência |
| **Velocidade** | Imagens WebP, lazy load, score PageSpeed > 90 |
| **Social proof acima do fold** | "+1.200 casos" visível no hero — reduz bounce |
