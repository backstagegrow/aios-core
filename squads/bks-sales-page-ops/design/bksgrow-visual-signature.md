# BKSGrow Visual Signature

## Core Principles
- Premium editorial typography with strong hierarchy
- Layered atmospheres (matte + glow balance)
- Intentional asymmetry in key sections
- Motion as narrative, not decoration

## Signature Elements
- Hero with depth layers
- Proof section with contrast rhythm
- CTA states with premium micro-interactions
- Carousel/gallery with smooth transitions and clear controls

## Avoid
- Flat generic blocks
- Random gradients without hierarchy
- Repetitive stock layout patterns
